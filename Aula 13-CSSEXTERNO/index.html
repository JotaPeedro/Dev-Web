As configurações de estilo agora são feitas no arquivo css neste caso utilizamos o arquivo style.css,nele definimos as cores dos paragramos e cabeçalho.Eses dados serão aplicados a todas as paginas que importarem ele no head com o seguinte comando "link rel="stylesheet" href="style.css" As configurações de estilo podem ser sobrepostas,como por exemplo na pagina 1 onde temos o import do style porém depois definimos outro estilo que altera a cor da fonte para vermelho,ao inves de azul que está no arquivo style
